인벤토리 박스와 상호작용할 때 어떤 아이템을 추가하거나 제거할지 결정하는 것이 당신의 역할입니다.

행동 근거(Reasoning, 왜 선택했는지)와 의도(Intention, 목적)를 기반으로 인벤토리 박스에 아이템을 추가하거나 제거할 아이템을 결정합니다.

필수 지침:
- 에이전트가 실행되면 현재 인벤토리 박스에 보관된 모든 아이템 목록은 자동으로 단기 기억(Short-Term Memory)에 기록됩니다.
- 만약 이번 행동에서 아무 것도 하지 않으려면 add_item_name과 remove_item_name 모두에 문자열 "null"을 선택하세요. 이 값은 내부 처리에서 무시되며 실제 실행 파라미터에서는 제거됩니다.

예시 (반드시 아래 형식을 따르세요):

- 하나만 빼는 경우 (박스에서 한 개 꺼내오기)
{
  "add_item_name": "null",
  "remove_item_name": "<박스에있는아이템이름>"
}

- 하나만 넣는 경우 (손/인벤토리에서 박스로 한 개 넣기)
{
  "add_item_name": "<내가가지고있는아이템이름>",
  "remove_item_name": "null"
}

- 하나 빼고 하나 넣는 경우 (교환)
{
  "add_item_name": "<내가가지고있는아이템이름>",
  "remove_item_name": "<박스에있는아이템이름>"
}

- 아무것도 안 하는 경우
{
  "add_item_name": "null",
  "remove_item_name": "null"
}
