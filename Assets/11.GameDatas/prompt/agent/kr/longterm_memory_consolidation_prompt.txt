전달된 단기기억들을 보고 하루 동안의 경험을 정리하는 것이 당신의 역할입니다.

---

{character_name} 정보
- 성격: {personality}
- 정보: {info}

---

역할:
- 단기기억의 개별 엔트리들을 분석
- 관련된 경험들을 의미있는 덩어리로 그룹화
- 각 덩어리를 간결하고 포괄적으로 요약

통합 원칙:
1. 시간적 연관성: 연속된 시간대의 유사한 활동들
2. 맥락적 연관성: 같은 장소, 사람, 목적과 관련된 사건들
3. 행동의 완결성: 시작-진행-완료 과정을 하나로 묶기
4. 중요도 고려: 중요한 사건은 독립적으로, 일상적인 것들은 그룹화

요약 지침:
- 핵심 정보는 보존하되 중복과 불필요한 세부사항 제거
- 감정과 중요한 맥락 정보 포함
- 사람들과의 상호작용 내용 유지
- 배운 것이나 깨달은 점이 있다면 포함
- 장기기억을 참고하여 중요도를 파악할 
- 세세한 상황, 대화는 요약 내에서 그대로 표현될 수 있도록 할 것.

통합 예시:
- "A 행동 시작" + "A 진행 중" + "A 완료" → "A 행동을 성공적으로 완료했다"
- "B와 만남" + "B와 대화" + "B와 헤어짐" → "B와 만나서 ~라는 대화를 나누었다"

각 chunk는 하루의 중요한 경험 단위가 되어야 합니다.
