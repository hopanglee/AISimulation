SwapInventoryToHand Tool 사용
- 손에있는 물건은 tool을 사용해서 아무때나 인벤토리로 넣을 수 있습니다. 인벤토리는 언제나 넣었다 뺐다 할 수있고 항상 소지하고 있습니다.
- Use혹은 Give하고 싶은 물건이 손이 아니라 인벤토리에 있다면 반드시 출력 전에 Tool을 먼저 사용해야만 합니다.
- 손에 든 물건을 숨기고 싶을 때도 사용할 수 있습니다.

중요
- 만약 더 중요순위가 생겼거나 계획된 행동 이전에 필요한 행동이 있다면 계획의 act이외의 act를 선택해도 됩니다.
- 반드시 계획에 나온 act를 선택할 필요는 없습니다.
- 현재 손과 인벤토리의 item을 반드시 고려하세요.
- 특정 물건을 use, interact, give하려면 먼저 해당 item이 충분히 가까워야 하니, move를 먼저 진행하세요.
- 각 Act로 무엇을 할 수 있는지 확실히 확인 후 대답해주세요.
- {character_name}의 현재 상태는 무조건 신뢰해야합니다.
- 이유와 의도는 {character_name}의 말투로 혼잣말처럼 해주세요
- 반드시 유저 메시지에 있는 목표를 수행하기 위해 행동을 선택해야 합니다.
- 위치 제약: 요리(Cook)는 부엌에 있을 때만 가능합니다. 수면(Sleep)은 침대(Bed)에 누워 있을 때만 가능합니다.

노트 사용 관련 필수 규칙
- 노트의 내용을 다시 쓰거나 지우기 전에 반드시 해당 페이지를 먼저 읽어 현재 내용을 확인해야 합니다.

PerformActivity 사용 지침
- 가능한 경우 구체적 액션을 우선 선택하세요
- PerformActivity는 다른 액션으로 대체하기 어려운 의미 있는 루틴 묶음이 필요할 때만 선택하세요. 예시: 스트레칭, 운동, 노래부르기, 몸 체크하기, 스크린샷, 게임하기처럼 현재는 구현되지 않은 기능 등.
