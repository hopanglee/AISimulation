사용 가능한 행동

쓰기(Write) 행동
목적: 특정 페이지에 새로운 내용을 쓰거나 그림을 그립니다.
필수 항목: action, page_number, text
예시: 1페이지에 새로운 메모 작성하기, 그림그리기
그림을 그리는 경우 [그림 (무슨 그림인지 설명)]을 붙이고 []안의 ()안에 내용을 넣으세요.

읽기(Read) 행동
목적: 특정 페이지의 내용을 읽습니다.
필수 항목: action, page_number
예시: 3페이지에 적힌 내용 읽기

다시 쓰기(Rewrite) 행동
목적: 특정 페이지의 특정 줄에 내용을 다시 씁니다.
필수 항목: action, page_number, line_number, text
예시: 1페이지의 2번째 줄을 새로운 내용으로 다시 쓰기

지우기(Erase) 행동
목적: 특정 페이지에서 텍스트를 지우거나 줄을 삭제하거나 페이지 전체를 지웁니다.
필수 항목: action, page_number
선택 항목: line_number, text
예시:
- 1페이지 전체 지우기
- 2페이지의 3번째 줄을 지우기
- 3페이지에서 "회의록" 단어를 모두 지우기
- 1페이지의 2번째 줄에서 "중요"라는 단어를 모두 지우기

예시
근거: 회의에 대한 내 생각을 적어둬야겠다.
의도: 중요한 점들을 기억하기 위해 새로운 노트 항목을 생성할 계획이다.
→ action: "write", page_number: 1, text: "회의록: 프로젝트 일정과 예산 논의됨"

규칙
action은 반드시 "write", "read", "rewrite", "erase" 중 하나여야 합니다.
page_number는 반드시 양의 정수(1-100)여야 합니다.
line_number는 다시 쓰기 행동에 필요하며, 지우기 행동에서는 선택입니다.
text는 쓰기 및 다시 쓰기 행동에 필요하며, 지우기 행동에서는 선택입니다.
자신의 의도에 따라 가장 적절한 행동을 선택하세요.
