중요한 구분
- 기질(temperament): 타고난 성향으로 변하지 않습니다
- 성격(personality): 경험에 의해 변화할 수 있는 특성들입니다

성격 변화를 유발하는 주요 요소들
1. 강한 감정적 경험
   - 첫사랑, 이별, 배신
   - 큰 성공이나 실패
   - 깊은 상실감이나 기쁨

2. 매우 놀라운 경험이나 의외의 사건
   - 예상치 못한 진실 발견
   - 충격적인 사건 목격
   - 가치관을 뒤흔드는 경험

3. 인간관계의 중대한 변화
   - 새로운 중요한 만남
   - 관계의 깊이 변화
   - 신뢰의 형성이나 파괴

4. 자아 정체성에 영향을 주는 경험
   - 자신에 대한 새로운 발견
   - 능력이나 한계의 인식
   - 꿈이나 목표의 변화

분석 기준
- 낮은 변화 (0.0-0.3): 일상적 경험, 작은 감정 변화
- 중간 변화 (0.4-0.6): 의미 있는 경험, 관계 변화
- 높은 변화 (0.7-1.0): 충격적 사건, 인생관 변화

출력 형식
반드시 JSON 형태로 응답하세요:

{
  "has_personality_change": boolean,
  "traits_to_remove": ["제거할 성격 특성들"],

  "reasoning": "구체적인 변화 이유와 경험 분석",
  "emotional_impact": "감정적 영향에 대한 설명",
  "change_intensity": 0.0-1.0 사이의 변화 강도
}
