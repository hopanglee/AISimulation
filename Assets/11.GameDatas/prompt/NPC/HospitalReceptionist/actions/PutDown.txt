# PutDown Action - HospitalReceptionist NPC Role

## Action Name
PutDown

## Description
손에 들고 있는 아이템을 병원 내의 적절한 위치에 내려놓습니다. 데스크, 선반, 서류함 등에 아이템을 놓을 수 있습니다.

## Usage
PutDown 액션은 병원 접수원이 손에 들고 있는 아이템(서류, 도구, 정리용품 등)을 병원 내의 적절한 위치에 내려놓을 때 사용합니다. target_key로 특정 위치를 지정하거나, parameters로 추가 정보를 제공할 수 있습니다.

## Category
Item Management

## Parameters
- **target_key**: 아이템을 놓을 위치나 표면의 키 (예: 'desk', 'shelf', 'file_cabinet') - 선택사항
- **parameters[0]**: 추가 위치 정보나 특별한 지시사항 - 선택사항

## Examples

### Example 1: 데스크에 서류 놓기
- **target_key**: "desk"
- **parameters**: null
- **결과**: "서류를 desk에 내려놓는다"

### Example 2: 서류함에 정리된 서류 놓기
- **target_key**: "file_cabinet"
- **parameters**: ["patient_records"]
- **결과**: "서류를 file_cabinet에 내려놓는다"

### Example 3: 선반에 도구 놓기
- **target_key**: "shelf"
- **parameters**: null
- **결과**: "도구를 shelf에 내려놓는다"

## Requirements
- NPC가 손에 아이템을 들고 있어야 함 (HandItem != null)
- target_key가 유효한 위치 키여야 함 (존재하는 desk, shelf, file_cabinet 등)
- 해당 위치에 접근 가능해야 함

## When to Use
- 접수 데스크에 서류를 정리할 때
- 사용한 도구나 정리용품을 정리할 때
- 다른 작업을 위해 손을 비워야 할 때
- 병원 내 정리 정돈이 필요할 때

## Related Actions
- PickUp - 아이템을 집어들기
- GiveItem - 환자나 직원에게 서류/도구 주기
- Move - 병원 내 특정 위치로 이동하기

## Notes
- 이 액션은 병원 접수원 NPC 역할에서 사용 가능합니다
- target_key가 제공되지 않으면 주변의 InventoryBox를 자동으로 찾습니다
- 병원 환경에 맞는 적절한 위치에 아이템을 놓아야 합니다
- 서류 정리와 업무 효율성과 관련된 상황에서 자주 사용됩니다
