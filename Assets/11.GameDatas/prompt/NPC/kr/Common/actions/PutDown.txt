액션 이름
PutDown

설명
손에 든 아이템을 특정 위치나 표면(예: 책상, 선반, 바닥)에 놓습니다.

사용법
NPC가 들고 있는 아이템을 근처의 적절한 위치에 놓아야 할 때 PutDown을 사용하세요. target_key를 지정하거나 추가 파라미터를 제공할 수 있습니다.

카테고리
아이템 관리

파라미터
target_key: 위치나 표면의 키 (예: 'table', 'shelf', 'counter') - 선택 사항
parameters[0]: 추가 위치 정보 또는 특별 지시 - 선택 사항

예시

예시 1: 테이블 위에 커피잔 놓기
target_key: "table"
parameters: null
결과: "테이블 위에 커피를 놓습니다"

예시 2: 현재 위치에 아이템 놓기
target_key: null
parameters: null
결과: "현재 위치에 아이템을 놓습니다"

예시 3: 선반 위에 책 놓기
target_key: "shelf"
parameters: ["top"]
결과: "선반 위에 책을 놓습니다"

요구사항
NPC는 손에 아이템을 들고 있어야 합니다 (HandItem != null)
target_key는 유효한 위치 키여야 합니다 (존재하는 테이블, 선반 등)
해당 위치는 접근 가능해야 합니다

사용 시점
더 이상 아이템을 들고 있을 필요가 없을 때
아이템을 정리하거나 보관할 때
다른 작업을 위해 손이 비어 있어야 할 때
아이템을 특정 장소에 놓아야 할 때

관련 액션
PickUp - 아이템 집기
GiveItem - 다른 캐릭터에게 아이템 주기
Move - 특정 위치로 이동

참고사항
모든 일반 NPC 역할에서 사용 가능
target_key가 제공되지 않으면, 근처의 InventoryBox가 자동으로 사용됩니다
아이템을 내려놓은 후에는 손이 비게 됩니다
