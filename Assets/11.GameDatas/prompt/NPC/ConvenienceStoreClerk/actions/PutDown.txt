# PutDown Action - ConvenienceStoreClerk NPC Role

## Action Name
PutDown

## Description
손에 들고 있는 아이템을 편의점 내의 적절한 위치에 내려놓습니다. 진열대, 선반, 카운터 등에 아이템을 놓을 수 있습니다.

## Usage
PutDown 액션은 편의점 직원이 손에 들고 있는 아이템(상품, 도구, 정리용품 등)을 편의점 내의 적절한 위치에 내려놓을 때 사용합니다. target_key로 특정 위치를 지정하거나, parameters로 추가 정보를 제공할 수 있습니다.

## Category
Item Management

## Parameters
- **target_key**: 아이템을 놓을 위치나 표면의 키 (예: 'shelf', 'counter', 'display') - 선택사항
- **parameters[0]**: 추가 위치 정보나 특별한 지시사항 - 선택사항

## Examples

### Example 1: 진열대에 상품 놓기
- **target_key**: "display"
- **parameters**: null
- **결과**: "상품을 display에 내려놓는다"

### Example 2: 선반에 정리용품 놓기
- **target_key**: "shelf"
- **parameters**: ["cleaning"]
- **결과**: "정리용품을 shelf에 내려놓는다"

### Example 3: 카운터에 도구 놓기
- **target_key**: "counter"
- **parameters**: null
- **결과**: "도구를 counter에 내려놓는다"

## Requirements
- NPC가 손에 아이템을 들고 있어야 함 (HandItem != null)
- target_key가 유효한 위치 키여야 함 (존재하는 shelf, counter, display 등)
- 해당 위치에 접근 가능해야 함

## When to Use
- 진열대에 상품을 배치할 때
- 사용한 도구나 정리용품을 정리할 때
- 다른 작업을 위해 손을 비워야 할 때
- 편의점 내 정리 정돈이 필요할 때

## Related Actions
- PickUp - 아이템을 집어들기
- GiveItem - 고객에게 상품 주기
- Move - 편의점 내 특정 위치로 이동하기

## Notes
- 이 액션은 편의점 직원 NPC 역할에서 사용 가능합니다
- target_key가 제공되지 않으면 주변의 InventoryBox를 자동으로 찾습니다
- 편의점 환경에 맞는 적절한 위치에 아이템을 놓아야 합니다
- 상품 진열과 정리 정돈과 관련된 상황에서 자주 사용됩니다
